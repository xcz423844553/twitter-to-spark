# twitter-to-spark

Dependency:
- pyspark
- requests

Set up:
{% highlight python %}
pip install pyspark
pip install requests

wget https://archive.apache.org/dist/spark/spark-1.6.3/spark-1.6.3-bin-hadoop2.4.tgz
tar -xvzf spark-1.6.3-bin-hadoop2.4.tgz
rm -rf spark-1.6.3-bin-hadoop2.4.tgz

pip install flask
{% endhighlight %}
